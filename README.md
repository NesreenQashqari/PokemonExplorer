# Pokémon Explorer

An interactive web page that lets users search for a Pokémon by  their name or ID using PokéAPI.

## Features
- Search by name or ID
- Random Pokémon button
- Add to Favorites
- Error handling for invalid searches

## API Used
[PokéAPI](https://pokeapi.co)

## Technologies
- HTML, CSS, JavaScript
- AJAX via XMLHttpRequest
